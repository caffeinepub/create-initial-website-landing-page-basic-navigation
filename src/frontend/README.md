# Archiv Agency - Frontend

A modern travel agency website built with React, Vite, and Tailwind CSS, deployed on the Internet Computer.

## üöÄ Quick Start (VS Code)

### Prerequisites

- **Node.js** v18 or later ([Download](https://nodejs.org/))
- **npm** (comes with Node.js) or **pnpm** v8+
- **VS Code** ([Download](https://code.visualstudio.com/))

### Installation & Development

1. **Open the project in VS Code:**
   ```bash
   code .
   ```

2. **Open integrated terminal** (`` Ctrl+` `` or `View > Terminal`)

3. **Navigate to frontend directory:**
   ```bash
   cd frontend
   ```

4. **Install dependencies:**
   ```bash
   npm install
   ```
   
   Or with pnpm:
   ```bash
   pnpm install
   ```

5. **Start development server:**
   ```bash
   npm run dev
   ```
   
   Or with pnpm:
   ```bash
   pnpm run dev
   ```

6. **Open in browser:**
   - The terminal will show: `Local: http://localhost:3000/`
   - Press `Ctrl+Click` (Windows/Linux) or `Cmd+Click` (Mac) on the URL
   - Or manually open: http://localhost:3000

### Available Scripts

From the `frontend/` directory:

- **`npm run dev`** - Start development server with hot reload (port 3000)
- **`npm run build`** - Create production build in `dist/` folder
- **`npm run preview`** - Preview production build locally
- **`npm run lint`** - Check code for errors
- **`npm run lint:fix`** - Auto-fix linting issues
- **`npm run format`** - Format code with Prettier
